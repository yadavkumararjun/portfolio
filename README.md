# Arjun Kumar Yadav - E-Portfolio

A professional, modern, and responsive e-portfolio website built with React, TypeScript, and Vite. Showcasing my skills, projects, experience, and contact information.

## 🚀 Live Demo

[Visit Portfolio](https://yadavkumararjun.github.io/portfolio)

## ✨ Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient-based design with smooth animations
- **TypeScript**: Fully typed React components for better code quality
- **Performance Optimized**: Fast build times with Vite
- **CI/CD Ready**: Automated GitHub Actions deployment to GitHub Pages
- **Accessibility**: Semantic HTML and keyboard navigation support
- **Dark Theme**: Eye-friendly dark theme with vibrant accent colors

## 📋 Sections

1. **Hero Section**: Engaging introduction with call-to-action buttons
2. **About**: Personal background, education, and professional summary
3. **Skills**: Technical skills with proficiency levels
4. **Experience**: Timeline of education, achievements, and projects
5. **Projects**: Featured and other projects with descriptions
6. **Contact**: Contact form and multiple ways to get in touch
7. **Footer**: Quick links and social connections

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/yadavkumararjun/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## 🏗️ Build

To build the project for production:

```bash
npm run build
```

The optimized build will be created in the `dist/` folder.

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint
- `npm run type-check` - Run TypeScript type checking

## 🚀 Deployment

The portfolio is automatically deployed to GitHub Pages using GitHub Actions whenever you push to the main branch.

### Manual Deployment

To deploy manually:

```bash
npm run build
# Push the build to GitHub Pages
```

The workflow file (`.github/workflows/deploy.yml`) handles automatic deployment.
